# JavaScript

## 简介

+ 使用：必须在“<script> </script>”之间使用
+ 函数可以在被调用的时候被执行
+ 可以把JavaScript脚本放在body或者head中
+ 也可以放在外部文件中，在script中声明调用

外部调用分离了html和JavaScript，更加易于维护和阅读，已缓存JavaScript文件可以快速加载

它是一种编程语言，和html的显示与css的规则不同，它有运行逻辑与变量，可以用于调用html和css

- 表达式：变量和运算符的组合，最后结果是一个值
- 关键字var声明全局作用域，在函数内就是局部
- 关键词let在块内声明局部作用域
- 关键词const定义不能被重新赋值的变量
- 注释：使用//或者/**/
- 标识符：首字符是下划线字母或者$，连串的字符可以是下划线字母数字或者$
- 对大小写是敏感的
- 分割单词使用驼峰式
- 字符集：Unicode

### 数据类型

JavaScript有动态数据类型

- 数值：数字    科学计数法：1.4443e10
- 字符串值：”文本“
- 布尔值： true or false
- 数组：[项目1，项目2，项目3]
- 对象：{name:"zhou",age:"20"}

tyoeof：运算符，返回数据类型

undefined：值，typeof返回没有值的变量

空值：有值，有类型，但是是空

Null: nothing，被当成对象

null和undefined值相等，但是类型不相等

原始数据和复杂数据：原始数据没有额外属性、方法单一，复杂数据有function或者objection等
